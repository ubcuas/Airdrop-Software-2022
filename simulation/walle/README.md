##  2. <a name='Runsimulation'></a>Run simulation

Click File -> Open World, Navigate to the repository location and open the *empty.wbt*

###  2.1. <a name='Command'></a>Command

mode change with  _

####  2.1.1. <a name='Manualmode'></a>Manual mode

| command | function   |
| ------- | ---------- |
| W       | go forward |
| A       | turn left  |
| S       | turn right |
| D       | go back    |
| Q       | full stop  |
